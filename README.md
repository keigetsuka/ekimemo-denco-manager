# 駅メモ！でんこ管理アプリ

「駅メモ！」の所持でんこを管理するWebアプリケーションです。HTML、CSS、JavaScriptのみで構成された静的アプリケーションで、ローカル環境で動作します。

## 機能概要

### 基本機能
- **でんこの所持状況管理**: 285体のでんこの所持・未所持をチェックボックスで管理（※No.0「のぞみ」は除外）
- **レベル情報の入力・管理**: 所持でんこのレベル（1～80）を記録
- **クラス情報の入力・管理**: 所持でんこのクラス（1～12）を記録
- **でんこ情報の表示**: 名前、タイプ、属性、スキル名、スキル効果を表示
- **完全なデータベース**: オリジナルでんこ163体・エクストラでんこ123体の全情報を収録

### フィルタ・検索機能
- **名前検索**: でんこ名での部分一致検索
- **タイプフィルタ**: トリックスター、サポーター、ディフェンダー、アタッカー
- **属性フィルタ**: heat、cool、eco
- **所持状況フィルタ**: 所持、未所持
- **レベル範囲フィルタ**: 指定したレベル範囲での絞り込み
- **クラス範囲フィルタ**: 指定したクラス範囲での絞り込み

### ソート機能
- **番号順**: でんこのID順（昇順・降順）
- **名前順**: でんこ名の五十音順（昇順・降順）
- **レベル順**: 所持でんこのレベル順（昇順・降順）
- **クラス順**: 所持でんこのクラス順（昇順・降順）

### データ管理機能
- **自動保存**: 変更内容は自動的にローカルストレージに保存
- **レベル分布グラフ**: 所持でんこのレベル分布を視覚的に表示
- **データエクスポート**: 所持データをJSONファイルとしてダウンロード
- **データインポート**: JSONファイルから所持データを読み込み

### 統計情報
- 総でんこ数、所持数、未所持数、平均レベル、平均クラスをリアルタイム表示

### ねこぱんち計算機（`nekopanchi.html`）
- **獲得経験値・到達レベルの計算**: 現在のレベルとレベル内経験値から、ねこぱんち使用後の状態をシミュレート
- **ねこぱんち種別**: 弱（150）/ 中（250）/ 強（400）/ ♪（100,000）/ イベント（300,000）exp
- **経験値ボーナス**: 誕生日（+25%）/ フィルム（+30%）/ 周年フィルム（+70%）/ 土日（+150%）/ レベルグングン（+120%）
- **ボーナス計算式**: `基本exp + 基本exp × (ボーナス率の合計)`（複数ボーナスは加算）
- **経験値テーブル**: `wiki/でんこ能力値一覧表.html` 由来のレベル別必要経験値（非累積）を使用
- **結果表示**: 獲得経験値、最終レベル、レベルアップ数、使用前後の状態、経験値進捗バー

## 使用方法

### ⚠️ 重要: HTTPサーバー経由でのアクセスが必要です

このアプリケーションは JSON ファイルを動的に読み込むため、**必ずHTTPサーバー経由でアクセスしてください**。

### 1. 推奨: HTTPサーバーでの起動
```bash
cd /home/sleach/ekimemo-denco-manager
python3 -m http.server 8080
# でんこ管理: http://localhost:8080
# ねこぱんち計算機: http://localhost:8080/nekopanchi.html
```

### 2. 他のHTTPサーバーを使用する場合
```bash
# Node.jsのhttp-serverを使用
npx http-server -p 8080

# Pythonの場合
python -m http.server 8080
```

### 3. ❌ 直接ファイルを開く場合の問題
- `index.html` や `nekopanchi.html` を直接ブラウザで開くと、CORS制限により JSON が読み込めません
- エラーメッセージが表示され、データが正しく表示されません

### 4. でんこの管理
1. **所持状況の設定**: 各でんこの「所持」チェックボックスをクリック
2. **レベルの入力**: 所持でんこのレベル入力欄に1～80の数値を入力
3. **クラスの入力**: 所持でんこのクラス入力欄に1～12の数値を入力
4. **変更の保存**: 変更は自動的に保存されます

### 5. フィルタ・検索の使用
1. **名前検索**: 上部の検索欄にでんこ名を入力
2. **フィルタ設定**: タイプ、属性、所持状況のプルダウンから選択
3. **レベル範囲**: 最小・最大レベルを入力して範囲指定
4. **クラス範囲**: 最小・最大クラスを入力して範囲指定
5. **結果の確認**: 条件に一致するでんこのみが表示されます

### 6. ソート機能
1. **ソート項目**: 「並び順」プルダウンから項目を選択
2. **ソート方向**: 「昇順 ↑」「降順 ↓」ボタンをクリックして切り替え

### 7. データの管理
- **データをグラフ表示**: レベル分布グラフをポップアップで表示
  - オリジナル/エクストラ/全体の切り替え可能
  - 横軸：レベル、縦軸：でんこ数の棒グラフ
  - 統計情報（対象数、所持数、平均レベル）も表示
- **データをエクスポート**: JSONファイルとしてダウンロード
- **データをインポート**: 以前エクスポートしたJSONファイルを読み込み

### 8. ねこぱんち計算機の使用
1. **アクセス**: `http://localhost:8080/nekopanchi.html`、またはトップページの「ねこぱんち計算機」リンクから遷移
2. **レベル入力**: でんこの現在レベル（1～80）を入力
3. **経験値入力**: 現在レベル内の経験値を入力（次レベル必要値未満）
4. **ねこぱんち選択**: 使用するねこぱんちの種類を選択
5. **ボーナス選択**: 該当する経験値ボーナスにチェック（任意）
6. **結果確認**: 入力に応じて獲得経験値・到達レベルが自動計算されます

**ボーナス計算の例**（ねこぱんち♪ 100,000 exp、土日+レベルグングン）:
```
100,000 + 100,000 × (150% + 120%) = 370,000 exp
```

```json
  "requiredExp": [0, 400, 800, "..."]
}
```

## カスタマイズ

### でんこデータの追加・編集

**重要**: 現在のでんこデータはサンプルです。正確な情報を使用するには以下の手順で更新してください。

#### 1. 正確なでんこ情報の収集
以下のWikiページから正確な情報を収集してください：

**オリジナルでんこ（No.0～162、計163体）**
- 基本情報: wiki/顔画像・タイプ・属性・色・スキル名_オリジナルでんこ - 駅メモ！情報.html

**エクストラでんこ（No.1～123）**
- 基本情報: wiki/顔画像・タイプ・属性・色・スキル名_エクストラでんこ - 駅メモ！情報.html

#### 2. データ抽出スクリプトの使用
`extract_denko_data_v3.py` スクリプトを使用してWikiページから自動的にでんこデータを抽出できます。

```bash
cd /home/sleach/ekimemo-denco-manager
python3 extract_denko_data_v3.py
```

**スクリプトの特徴:**
- オリジナルでんこ163体とエクストラでんこ123体の情報を自動抽出
- 基本情報（名前、タイプ、属性、色）とスキル情報を正確にマッピング
- 外国語名でんこ（「・」「＝」「=」区切り）のファーストネーム抽出に対応
  - 例: 「エルミーヌ・ワロン」→「エルミーヌ」でスキル検索
  - 例: 「シャルロッテ＝フォン＝ハノーファー」→「シャルロッテ」でスキル検索
  - 例: 「アリス=カータレット」→「アリス」でスキル検索
- **特殊な外国語名でんこの例外処理**（2番目のネームを使用）
  - ジン・ティエン → ティエン
  - ツー・スイラン → スイラン
  - ロン・リンファ → リンファ
  - ハノイ・シュアン → シュアン
  - ナム・ソユン → ソユン
  - ピン・ユートン → ユートン
- スキルマッチング率（現行 `denko_data.json` ベース）: オリジナル93.3%（152/163件）、エクストラ50.4%（62/123件）

#### 3. JSONファイルの編集
`denko_data.json` を編集してでんこ情報を追加・変更してください。各でんこには以下の項目が必要です：
   - `id`: 一意の番号
   - `name`: でんこ名
   - `type`: タイプ（トリックスター、サポーター、ディフェンダー、アタッカー）
   - `attribute`: 属性（heat、cool、eco）
   - `color`: 色
   - `skill_name`: スキル名
   - `skill_effect`: スキル効果

### スタイルのカスタマイズ
- `styles.css` を編集してデザインを変更
- レスポンシブデザインに対応済み

## 技術仕様

- **対応ブラウザ**: モダンブラウザ（Chrome、Firefox、Safari、Edge）
- **データ保存**: ローカルストレージ（ブラウザ内）
- **ファイル形式**: HTML5、CSS3、ES6+ JavaScript
- **外部依存**: なし（純粋な静的ファイル）

## 注意事項

- データはブラウザのローカルストレージに保存されるため、ブラウザのデータを削除すると消失します
- 異なるブラウザやデバイス間でのデータ共有はできません
- でんこの画像は含まれていません
- でんこ情報は手動で収集したサンプルデータです
- **オリジナルでんこNo.0「のぞみ」は例外のため表示対象外です**
- ねこぱんち計算機はスキル効果による経験値増加には未対応です
- 経験値テーブルを更新する場合は `wiki/でんこ能力値一覧表.html` を更新後、`exp_table.json` を再生成してください
